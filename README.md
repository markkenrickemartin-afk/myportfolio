## 📌 Project Overview
This project is a personal portfolio website developed as a Laboratory Activity. It highlights my journey as a BSIT student, my technical skills, and my personal interests through a modern web interface.

## 🚀 Technologies Used
* **Backend:** Java (JDK 21+)
* **Framework:** Spring Boot 3.4.2
* **Template Engine:** Thymeleaf
* **Frontend:** HTML5, CSS3 (Modern Dark Theme)
* **Build Tool:** Maven

## 📂 Project Structure (5 Core Pages)
1. **Home:** Introduction, personal details, and profile photo.
2. **About Me:** Biography, my motivation for choosing BSIT, and a look at my life as a student.
3. **My College Journey:** Highlights of my academic experiences and challenges at NEUST.
4. **Skills & Interests:** A showcase of my technical stack (Java, MySQL, etc.) and hobbies (Chess/MLBB).
5. **Photo Gallery:** A collection of 10+ memorable moments with captions.

## 📸 Technical Requirements Met
- [x] Spring Boot MVC Architecture
- [x] Modern & Consistent UI Design
- [x] Responsive Layout using CSS Grid/Flexbox
- [x] Fragmented Navigation (shared across all pages)
- [x] 10+ High-Quality Images with Captions
